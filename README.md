# React + Redux Starter Kit

To help you kick start your  next react-redux project in no time. 
<hr>

![logo](assets/logo.png)

<hr>

### Setup and Run

* Clone this repo
* Type ```npm clean-commits // to start with a fresh git repo``` 
* Type ```npm install```
* Type ```npm run dev```
* In the browser,  type ```localhost:5000```
